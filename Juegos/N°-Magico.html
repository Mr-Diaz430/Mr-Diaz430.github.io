<Número Mágico (C++)/>

Bienvenido al apartado de juegos, En esta sección te mostraremos la codificación de un juego en particular, con la intención de que seas capaz de presenciar un poco de lo que es posible de realizar con un lenguaje de programación sin tener conocimientos avanzados. Puedes probar el juego desde tu entorno de desarrollo únicamente copiando el código del proceso.

Adivina el número!

El número mágico es un juego en que tienes que utilizar tu lógica para adivinar un número secreto que el ordenador escoge al principio del partido.

Como jugarlo

El número está formado por dígitos del 1 al 100; cada dígito aparece una vez como máximo. Tienes que adivinar el número con la menor cantidad de intentos posible, un intento consta de un número propuesto por el jugador y la respuesta del ordenador. Al finalizar el ordenador te informa cuantos intentos de tomo descubrir el número secreto.

Metodo Principal usando while


        int main() {
          srand(time(NULL));  // inicializar el generador de numeros aleatorios
          int numeroSecreto = rand() % 100 + 1;  // generar un numero aleatorio entre 1 y 100
          int intentos = 0;
          int numeroElegido = 0;
        
          std::cout << "Bienvenido a El Numero Magico!" << std::endl;
        
          while (numeroElegido != numeroSecreto) {
            std::cout << "Ingresa un numero entre 1 y 100: ";
            std::cin >> numeroElegido;
        
            if (std::cin.fail()) {
              std::cout << "Ingresa un numero valido." << std::endl;
              std::cin.clear();
              std::cin.ignore(10000, '\n');
              continue;
              }
        
            if (numeroElegido < 1 || numeroElegido > 100) {
               std::cout << "Ingresa un numero entre 1 y 100." << std::endl;
               continue;
               }
        
            intentos++; //sumador
        
            if (numeroElegido > numeroSecreto) {

              std::cout << "El numero es menor." << std::endl;

            } else if (numeroElegido < numeroSecreto) {

              std::cout << "El numero es mayor." << std::endl;

                } else {
                    system("cls"); 
                    std::cout <<"\n";
                    std::cout << "////////////////////////////////////////////////////////\n";  
                    std::cout << "//                                                    //\n"; 
                    std::cout << "//";           	
                    std::cout << "  Felicidades, adivinaste el numero en " << intentos << " intentos!";
                    std::cout << "  //\n";   
                    std::cout << "//                                                    //\n";                                                  
                    std::cout << "////////////////////////////////////////////////////////\n"; 
                }
                std:: cout << "\n";
              }
            return 0;
        }
