<Librerías Basicas (Java)/>

Bienvenidos a la sección de Librerías básicas, aquí encontrará algunas de las librerías más importantes/usadas y sus funciones más útiles para tener en cuenta a la hora de programar y afrontar diferentes tipos de ejercicios en Java.

Clase_Math{}

Esta clase representa la librería matemática de Java. Su constructor es privado, lo que nos permite crear instancias de la clase. La clase Math es una clase estática, eso quiere decir que sus funciones pueden ser utilizadas directamente desde la clase Math sin necesidad de crear una instancia de ella. Esta clase permite realizar operaciones matemáticas comunes, es importante tener en cuenta que los métodos de esta clase solo operan con tipos de datos primitivos (como int, double, etc.) y no con objetos. Algunos de los métodos más comunes incluyen:

Potencia y Raiz
Pow Potencia
sqrt Raiz Cuadrada
cbrt Raiz Cubica

Ejemplo de uso:

                        
      1       package math.potencia.y.raiz;
      2      /*
      3      *   @author DM
      4      */    
      5      public class MathPotenciaYRaiz {
      6                      
      7        /**
      8        * @param args the command line arguments
      9        */
     10        public static void main(String[] args) {
     11                               
     12      //Tener en cuenta que Pow, sqrt y cbrt solo funcionan con valores del tipo double.
     13    
     14         double x=25, r, r2, r3;
     15                                
     16         r = Math.pow(x, 2);   //Potencia. 
     17         r2 = Math.sqrt(x);   //Raiz cuadrada.                    
     18         r3 = Math.cbrt(x);   //Raiz cubica de X.  
     19                       
     21         
     22        System.out.println(x + " Elevado a " + 2 + " es: " + r);             
     23        System.out.println("La raiz cuadrada de: "+x+" es: "+r2);                    
     24        System.out.println("La raiz cubica de: "+x+" es: "+r3);
     25                  
     26       }   
     27      }
Mínimo y Máximo
min Devuelve el menor entre X e Y
max Devuelve el mayor entre X e Y

Ejemplo de uso:

                              
      1        package math.min.y.max;                      
      2                           
      3       /**  
      4       *  @author DM
      5       */ 
      6       public class MathMinYMax {  
      7
      8         /**  
      9         *  @param args the command line arguments
     10         */
     11         public static void main(String[] args) {
     12
     13             int X=2, Y=3, R, R2;    
     14
     15          R = Math.min(X, Y);   //Devuelve el mayor de x e y.    
     16          R2 = Math.max(X, Y);   //Devuelve el menor de x e y.
     17
     18          System.out.println("El numero mayor es: "+R2);
     19          System.out.println("El numero menor es: "+R);
     20         }
     21       }
Valor absoluto, aleatorio y redondeo
abs Valor absoluto
random Devuelve un número aleatorio entre 0 y 1
round Devuelve el entero más cercano

Ejemplo de uso:

                              
      1        package math.absoluto.aleatorio.y.redondeo;
      2                          
      3       /**
      4       * @author Lucas
      5       */
      6        public class MathAbsolutoAleatorioYRedondeo {
      7 
      8         /**
      9         * @param args the command line arguments
     10         */
     11           public static void main(String[] args) {
     12             double X=-2.5, Y=2.5, NRand, NRedon, R;
     13        
     14        
     15           R = Math.abs(X); //Retorna el valor absoluto de X.
     16        
     17           NRand = Math.random(); //Retorna un numero entre 0 y 1.
     18        
     19           NRedon = Math.round(Y); /*Retorna el entero mas cercano a Y.
     20                                                           (a partir de 0.5 redondea hacia un entero mayor)*/
     21        
     22        
     23           System.out.println("El valor absoluto de "+X+" es: "+R);
     24           System.out.println("Valor aleatorio entre 0 y 1: "+NRand); 
     25           System.out.println("El valor redondeado de "+Y+" es: "+NRedon); 
     26      }
     27  }
Grados y Radianes
toDegrees Grados a radianes
toRadians Radianes a Grados

Ejemplo de uso:

                              
      1        package math.grados.y.radianes;
      2
      3        /**
      4        * @author Lucas
      5        */
      6        public class MathGradosYRadianes {
      7                          
      8            /**
      9            * @param args the command line arguments
     10            */
     11             public static void main(String[] args) {
     12                    double grad=60, rad=1.6, R, R2;
     13                                   
     14              //Estas 2 funciones sirven unicamente con valores de tipo double.
     15                                  
     16               R = Math.toRadians(grad); //Retorna la conversión de grad a radianes.
     17                                 
     18               R2= Math.toDegrees(rad);  //Retorna la conversión de rad a grados. 
     19                                   
     20                                 
     21               System.out.println(grad+" grados es igual a: "+R+" radianes");
     22              System.out.println(rad+" radianes son igual a: "+R2+" grados");        
     23            }   
     24       }
