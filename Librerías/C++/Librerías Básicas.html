<Librerías Básicas (C++)/>

Bienvenidos a la sección de Librerías básicas, aquí encontrará algunas de las librerías más importantes/usadas y sus funciones más útiles para tener en cuenta a la hora de programar y afrontar diferentes tipos de ejercicios en C++.

<Iostream>

Es definitivamente la librería más utilizada, tiene muchas funciones sencillas que al incluirlas en nuestro código fuente generalmente evitamos tener que incluir más librerías. Está especializada en la lectura y escritura (input / output) de archivos y es exclusiva de C++.

Flujo de entrada y salida estándar:
Cout Flujo de salida estándar
Cin Flujo de entrada estándar

Ejemplo de uso:

                
      1    #include <iostream>	
      2    int main(){
      3    int B;
      4                        
      5    std::cin>> B; /*Ingresa un valor para B desde la consola, funciona como entrada de datos (Input).*/
      6    
      7    std::cout<< B; /*Muestra el valor de B en la consola, funciona como salida de datos (Output).*/
      8    
      9    return 0;
     10   }

<cstdlib> (stdlib.h)

cstdlib (Biblioteca de utilidades generales estándar de C) es un encabezado que define varias funciones de propósito general, incluida la gestión de memoria dinámica, la generación de números aleatorios, la comunicación con el entorno, la aritmética de enteros, la búsqueda, la clasificación y la conversión. Es compatible con C en donde se conoce como stdlib.h.

Entorno:
system("pause") Pausa la ejecución del codigo
sytem("cls") Limpia consola de todo el texto previo

Ejemplo de uso:

      2    #include <cstdlib>
      3

     10             
     11     System(" ") tiene muchas funcionalidades tanto practicas como estéticas, pero en este caso solo veremos algunas funciones prácticas 
     12    para nuestros programas. Si desea ver las funciones estéticas de system(" ") puede trasladarse a la sección librerías estéticas (C++).*/
     13             
     14    int main(){

     16 
     17    //Realizamos un simple bucle para poner en práctica la utilidad de system.

Generación de secuencias pseudoaleatorias:
rand Generación de números aleatorios
srand Inicializa el generador de números aleatorio

Ejemplo de uso y problemas frecuentes:

                
      1     #include <iostream>	
      2    #include <cstdlib>
      3    #include <ctime>
      4              
      5    using namespace std;
      6              
      7    int main(){
      8               
      9   // srand(): Inicializa el generador de números aleatorio.  
     10               
     11    // rand(): Retorna un número integral de manera pseudo-aleatorio entre 0 y un valor máximo (este valor máximo está dado por la  
     12                      constante RAND_MAX que viene incluida dentro de la librería cstdlib).  
     13               
     14    cout<< RAND_MAX <<endl; /*RAND_MAX: Es una constante que determina el valor máximo del generador de números  
     15                                                     "aleatorios" (el valor de esta constante dependerá del compilador).*/ 
     16                
     17    /*------------------------------------------------------------------------------------------------------
     18    Problema 1: 
     19                
    20     Al usar rand() este generará un número de forma "aleatoria", número el cual al terminar la ejecución de nuestro programa NO 
    21      cambiará, lo que perjudica nuestras intenciones de usar rand() para generar números de forma aleatoria.
    22               
    23     Solución:
    24     Esto sucede porque rand() no está cambiando la serie que genera, por lo que para obtener números diferentes en cada ejecución  
    25     primero debemos establecer un punto de partida o un valor inicial.
    26                
    27     Para ello usaremos nuestra función srand() para darle un valor inicial tal que nuestro rand() cambie con cada 
    28     ejecución: El valor inicial/parámetro que le daremos a srand() será la variable time(0) que viene integrada en la librería  
    29    <ctime> (por lo que también debemos agregarla en la parte superior de nuestro código). Time() nos devuelve un valor de tiempo que variara dependiendo la hora de nuestra PC.
    30     Con esto nos aseguraremos de que el valor de rand() nunca sea el mismo.
    31     */          
    32              
    33     srand((unsigned)time(0)); /*Recibe un valor inicial sin signo por parte de la variable time(). 
    34                                                        Otra forma de escribirlo podría ser: srand(time(NULL)). */
    35       
    36      cout << rand()  <<endl; //Selecciona un número entre 0 y RAND_MAX.             
    37     /*-------------------------------------------------------------------------------------------------
    38                
    39     Problema 2: Generar números aleatorios entre rangos.
    40                
    41     A) Si usted ha probado el código escrito anteriormente se dará cuenta de que los números que rand() suele retornar son un poco grandes,
    42        por lo que si por alguna razón desea establecer un valor máximo de números aleatorios que se puedan generar, lo podemos hacer 
    43        con la operación matemática MOD(%):
    44                 
    45    Solucion:  */
    46
    47     cout << (rand() % 11)  <<endl; //Seleccionará un número entre 0 y 10
    48         
    49    /* 
    50       B) De igual manera que podemos limitar el valor máximo podemos establecer un valor mínimo, lo que debemos hacer es sumarle al 
    51           (rand() % Número-Máximo) un número desde el cual queremos que inicie nuestro generador de números "aleatorios". Pero al hacer esto nuestro  
    52           Número-Máximo deberá cambiar de la siguiente manera:     
    53                          
    54     1. cout << (rand() % NumeroMaximo)  <<endl; //La variable NumeroMaximo limita el valor máximo de números - 1 que rand() pueda generar.
    55                     
    56     2. cout << (rand() % NumeroMaximo) + NumeroMinimo  <<endl; // Al darle un Valor mínimo a nuestro generador, 
    57                                       nuestro NumeroMaximo tiene que pasar a ser la diferencia (resta) entre NumeroMaximo y NumeroMinimo
    58                                       para poder mantener el rango de valores que nosotros deseamos tener.
    59                     
    60     3. cout << (rand() % [NumeroMaximo - NumeroMinimo]) + Numero-Minimo  <<endl; //Esto nos dará como resultado un generador que se mantenga en el intervalo de números que deseemos.
    61      
    62            Esto sucede porque al agregar una suma con el valor mínimo no solo estamos modificando el valor mínimo de rand(), que recordemos que es 0,                                                                   
    63            sino también modificaremos el resultado del módulo por lo cual es importante tener en cuenta todo lo visto. A continuación se realizará lo mismo
    64             , pero  reemplazando las variables Numero-Maximo y Numero-Minimo por números.
    65               
    66                 
    67                EJ: Nuestra meta es conseguir un rand() con un rango de números de 10 a 20.  */
    68                 
    69               cout << (rand() % 21)  <<endl;  //Selecciona un número entre 0 y 20.
    70                   
    71                cout << (rand() % 21) + 10  <<endl; //Selecciona un número entre un rango de números no deseado con un Valor-Minimo: 10.
    72                  
    73               cout << (rand() % 11) + 10  <<endl; //Selecciona un número entre 10 y 20. Recuerda que el 11 surge de la diferencia entre 21 - 10.
    74                  
    75              return 0;  
    76              }
Aritmética entera:
abs Valor absoluto

Ejemplo de uso:

                
     1     #include <iostream>
     2    #include <cstdlib>
     3  
     4     using namespace std;
     5  
     6     //Este programa determina el valor absoluto de un número entero.
     7  
     8     int main(){
     9      int N=-10, R;
    10     
    11      //abs(número-entero): obtiene el valor absoluto de un número entero (int).
    12      
    13      R=abs(N);
    14      cout<<"\nEl valor absoluto de "<<N<<" es: "<<R<<endl; // R=10
    15    
    16      //Recordar que si usamos abs( ) con la libreria <cstdlib> este solo podra recibir valores enteros, de lo contrario
    17        dara error y no nos dejara ejecutar el codigo. Pero si declaramos la libreria <cmath> en nuestro codigo si podremos 
    18        usar valores flotantes en la funcion abs( ).
    19
    20        return 0;
    21     }

<cmath> (math.h)

Declara un conjunto de funciones principalmente para operaciones matemáticas y transformaciones. Se puede usar tanto en C++ como en C donde es conocida como math.h .

Funciones trigonométricas:
cos Calcula coseno
sin Calcula seno
tan Calcula tangente

Ejemplo de uso:

                
      1     #include  <iostream>
      2    #include  <cstdlib>
      3    #include  <cmath>
      4  
      5     using namespace std;
      6  
      7     /*Tener en cuenta que sin, cos y tan deben recibir parámetros en radianes por lo que primero debemos realizar la conversión de grados a radianes.*/
      8  
      9     int main(){
     10     float grad ,rad , PI=3.1415926536, seno, coseno, tangente;
     11     
     12     cout<<"Ingrese un angulo en grados: "; // le asignamos a nuestra variable grad el valor ingresado en grados
     13     cin>>grad;
     14    
     15     rad = grad * (PI / 180); // Mediante esta operación logramos transformar los grados ingresados anteriormente en radianes.
     16     
     17     
     18     coseno = cos(rad); // cos() retornará el coseno de un ángulo de x radianes. Y la guardamos en nuestra variable coseno.
     19     
     20     seno = sin(rad); // sin() retornará el seno de un ángulo de x radianes. Y la guardamos en nuestra variable seno.
     21   
     22     tangente = tan(rad); // tan() retornará la tangente de un ángulo de x radianes. Y la guardamos en nuestra variable tangente.
     23     
     24
     25     cout<<"\n El coseno es: "<<coseno;
     26     cout<<"\n El seno es: "<<seno;
     27     cout<<"\n la tangente es: "<<tangente<<endl<<endl;
     28   
     29     system("pause");
     30     return 0;
     31   }
Funciones de potencia:
pow Elevar a potencia
sqrt Calcular la raíz cuadrada
cbrt Calcular la raíz cubica

Ejemplo de uso:

                
      1      #include <iostream>
      2     #include <cstdlib>
      3     #include <cmath>
      4  
      5     using namespace std;
      6  
      7     int main(){
      8    
      9     float N, P, raizcuad, raizcub, potencia;
     10     
     11      cout<<"Ingrese un numero: ";
     12     cin>>N;
     13     
     14     cout<<"\nIngrese una potencia: ";
     15     cin>>P;
     16     
     17     raizcuad = sqrt(N); //sqrt() Calcula la raíz cúbica cuadrada de N.
     18     
     19     raizcub = cbrt(N); //cbrt() Calcula la raíz cúbica de N.
     20     
     21     potencia = pow(N , P); //pow() Realiza una potencia entre N (base) y P (exponente).
     22     
     23    system("cls");
     24   
     25    cout<<"La raiz cuadrada de "<<N<<" es: "<<raizcuad<<endl;
     26   
     27    cout<<"La raiz cubica de "<<N<<" es: "<<raizcub<<endl;
     28     
     29    cout<<N<<" elevado a "<<P<<" es: "<<potencia<<endl<<endl;
     30       
     31     system("pause");
     32     return 0;
     33   }

Librerías de C en C++

Esta es una lista con la mayoría de librerías que se utilizaban en C y que han sido convertidas para que puedan usarse también en C++. Para aclarar y como se mostró en ejemplos anteriores, las librerías en C usan una extensión .h al final del nombre, pero en C++ esta extensión se excluye y en su lugar se añade una c al inicio del nombre de la librería. Por ejemplo <math.h> (C) pasa a ser <cmath>(C++). 

                
  #include <cassert>        //Nos ayuda a encontrar las aserciones dentro de un programa. (especifica una condición que se espera que sea cierta).

  #include <cctype>         /*Declara un conjunto de funciones para clasificar y transformar caracteres. individuales. 
                                              EJ: Comprueba si un carácter es alfanumérico.*/
    
  #include <cerrno>          /*En ella se definen las macros que presentan un informe de error a través de códigos de error. 
                                              EJ: resultados de un parámetro fuera de una función de dominio, por ejemplo, sqrt (-1).*/

  #include <cfloat>           //Contiene parámetros de entorno, información sobre limitaciones y rangos para tipos reales.

  #include <ciso646>       /*Este encabezado define once constantes de macro con grafías alternativas para aquellos operadores de C++ que no son compatibles
                                               con el juego de caracteres estándar ISO646 EJ:  &&,  &=,  &,  |,  ~,  !,   !=,  ||,  |=,  ^  y  ^= */ .

  #include <climits>         //Contiene parámetros de entorno, información sobre limitaciones y rangos para tipos enteros.

  #include <clocale>        /*Contiene los prototipos de Las funciones, macros, y tipos para manipular y controlar varias opciones pertenecientes
                                                a la Localidad del sistema.*/
    
  #include <cmath>         //Contiene funciones para operaciones matemáticas y transformaciones.

  #include <csetjmp>      //Proporciona "saltos no locales": control de flujo que se desvía de la llamada sub-rutina habitual y la secuencia de retorno.

  #include <csignal>        //Contiene las funciones, macros, y tipos para crear y manipular señales del sistema.
  
  #include <cstdarg>       //Define macros para acceder a Los argumentos individuales de una lista de argumentos sin nombre.
  
  #include <cstddef>      /*Este encabezado define varios tipos implícitamente generados o usados por ciertas expresiones de Lenguaje.
                                             Ej: para definir macros como NULL y tipos de datos muy útiles como ptrdiff_t*/

  #include <cstdio>         //Biblioteca encargada de realizar operaciones de entrada/salida con printf y scanf.
  
  #include <cstdlib>        //Biblioteca de utilidades generales estándar.
  
  #include <cstring>       //Define varias funciones para manipular cadenas y arrays.
  
  #include <ctime>         //Contiene definiciones de funciones para obtener y manipular información de fecha y hora.
