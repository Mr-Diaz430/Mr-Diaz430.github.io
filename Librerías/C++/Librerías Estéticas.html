<Librerías Estéticas (C++)/>

Bienvenidos a la sección de Librerías Estéticas, en esta sección veremos distintas utilidades visuales que nos puede aportar la librería <cstdlib>. Como cambiar el color del fondo, texto y hasta colocar un título.

<cstdlib>

cstdlib (Biblioteca de utilidades generales estándar de C) es un encabezado que define varias funciones de propósito general. En esta sección desarrollaremos únicamente sus funciones estéticas. Si desea saber sobre las demás funciones de propósito general que contiene esta librería puede dirigirse a la sección de librerías básicas (C++).

Título
system ( "Title" ) Permite colocar un título en la consola

Ejemplo de uso:

                
      1    #include <iostream>
      2   #include <cstdlib>
      3   using namespace std;
      4   
      5   int main(){
      6
      7   /*Colocarle un título a nuestro programa por consola es muy útil y añade mayor profesionalismo a nuestros programas. Esto podemos
      8    lograrlo con tan solo una línea de código donde indicaremos dentro de la función system("") la etiqueta "Title". Y seguido (separado con un espacio)
      9    colocaremos lo que deseemos mostrar en nuestra ventana como título.*/
     10
     11    system( "Title C++: Linea de comandos profesional" );
     12
     13   system("pause");
     14   return 0;
     15   }
Cambiar colores
System( "Color" ) Cambia color del texto y consola

Ejemplo de uso:


      1    #include <iostream>
      2   #include <cstdlib>
      3   using namespace std;
      4
      5   int main(){
      6
      7   /*Para cambiar el color de la consola y del texto. Tenemos que usar la etiqueta "Color" dentro de system(""), y luego de dejar un espacio
      8   escribimos el código de los colores que deseemos usar: donde el primer número o letra indicará el color del fondo y el segundo número o letra indicará él
      9   color del texto. En el siguiente código puede observar cómo se implementa. */
     10          
     11    system("Color f0"); //Fondo Blanco - Texto Negro.
     12
     13   system("pause");
     14   return 0;
     15   }
     16
